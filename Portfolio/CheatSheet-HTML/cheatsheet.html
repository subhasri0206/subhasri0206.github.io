HTML - CheatSheet

HTML Introduction

  • HTML (HyperText Markup Language) is the standard language used to create web pages.
  • HTML is important because it tells your web browser (like Chrome, Firefox, or Safari) how to display the content on your webpage.
  • Every HTML document has a basic structure.

Document Structure

DOCTYPE Introduction

  • HTML Doctype is declared at the beginning of HTML document to inform browsers about the document type and version, ensuring correct rendering.
  • They arenâ€™t HTML tags but provide essential information.
  • A DOCTYPE declaration, appearing at the top of a web page, is required by HTML standards which ensures that browsers display the page as intended, adhering to the specified HTML version.

DOCTYPE Declaration

Declares the document type and version of HTML. It helps the browser to render the page correctly.

Syntax: <!DOCTYPE html>

Example Code: 


                        <!DOCTYPE html>
                        <html>
                            <head>
                                <title>Doctype Example</title>
                            </head>
                            < body >
                                <p>Hello, World!</p>
                            </body>
                        </html>

Rendered Output: The browser understands that this is an HTML5 document.

Anatomy of HTML

An HTML element typically consist of:

  • Opening Tag - The opening tag tells the browser where the element begins. It's enclosed in angle brackets (< >).
  • Content - The content is the actual text or other HTML elements that the tag is wrapping around.
  • Closing Tag - The closing tag tells the browser where the element ends. It's also enclosed in angle brackets but has a forward slash (/) before the element name.

Anatomy of an HTML Element

Syntax: <tagname> content </tagname>

Example Code: 


                        <p>This is a paragraph.</p>

Rendered Output: This is a paragraph.

Introduction to Body

  • The body tag defines the document's body.
  • The body element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • There can only be one body element in an HTML document.

Body Element

Contains the content of the HTML document.

Syntax:
<body>
...
</body>

Example Code: 


                        <!DOCTYPE html>
                        <html>
                            <head>
                                <title>Body Element Example</title>
                            </head>
                            <body>
                                <p>Hello, World!</p>
                            </body>
                        </html>

Rendered Output: Displays the content within the body element.

Forms

  • An HTML form is used to collect user input.

HTML Forms

HTML forms are used to collect user input.

Syntax: 


                        <form action="submit_url" method="post">
                            <input type="text" name="name">
                        </form>

Example Code: 


                        <!DOCTYPE html>
                        <html>
                            <head>
                                <title>Form Example</title>
                            </head>
                            <body>
                                <form action="/submit" method="post">
                                    <label for="name">Name:</label>
                                    <input type="text" id="name" name="name">
                                    <button type="submit">Submit</button>
                                </form>
                            </body>
                        </html>

Rendered Output: Form with a text input and submit button

Iframe

  • An HTML iframe is used to display a web page within a web page.
  • The HTML iframe tag specifies an inline frame.
  • An inline frame is used to embed another document within the current HTML document.

HTML Iframe

Defines an inline frame.

Syntax: <iframe src="url" width="" height=""></iframe>

Example Code: 


                        <!DOCTYPE html>
                        <html>
                            <head>
                                <title>Iframe Example</title>
                            </head>
                            <body>
                                <iframe src="https://example.com" width="300" height="200"></iframe>
                            </body>
                        </html>

Rendered Output: Inline frame displaying content from https://example.com

  • Global Attributes can be used with any HTML element

Common attributes that can be applied to any HTML element.

Syntax: <element global-attribute="value"></element>
